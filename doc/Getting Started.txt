What is CFS?
  - Customer management and relationships
  - Examples
    - Account management
    - Contact management
    - Sales pipeline management
    - Communications management
    - Case management
    - Document management
    - Call management
    - Project management
  - Why use CFS instead of something elese?
    - Open code
    - Integration with other products
    - Streamlines productivity
  
What language is CFS written in?
  - Java J2SE, J2EE
  - Servlets, JSP, Taglibs, JavaBeans
  - JDBC, JavaMail
  - SQL
  - XML
  
What programming methodology is used?
  - MVC
    - Layout
    - Modules (business rules)
    - JSPs (presentation)
    - JavaBeans
  - Database Independence
  - Iterative
  - Reusability
  - Sun coding conventions and documentation
  - Database conventions
  - HTML standards compatible with Internet Explorer 5+, Mozilla 1.0+

  - Hardware

How do I install ASPCFS?
  - Configure web and database server
  - Get a copy of the source code
  - Run the database install script
  - Run the web build/deploy script

What are the components and capabilities of ASPCFS?
  - Web Framework
    - Security
      - SSL
      - User can only access their own data, unless otherwise allowed
      - Single session sign-on
      - Auto-logout
      - Encrypted passwords
    - Roles
    - Permissions based on roles
    - Contact information
    - User information and hierarchy
    - Menus: Top-level, sub-menus, and container menus
    - Global Items: My Items, Recent Items
    - Global Search
    - Web-based XML API
    - Object Hook Manager
    - Synchronization
    - Modules
    - Pages
    - Connection Pool
    - System Preferences
    - Scheduled actions
    - Usage
  - Notification Manager
    - Email
    - Fax
  - Work Flow Manager
    - Multi-threaded
    - Standalone and integrated (hooks)
  - Data Import and Export (Reader/Writer)

How do I add a module to ASPCFS?
  - code module, extend com.darkhorseventures.cfsmodule.CFSModule
    - use a database connection in a module
  - code beans, extend GenericBean
    - auto-populate a JavaBean
    - sync a bean
  - code JSPs
    - html initPage.jsp  toHtml() toText() toHtmlValue()
    - beans
    - tag-libs
    - javascript
  - cfs-config.xml
  - cfs-modules.xml
  - cfs-container_menus.xml
  - cfs-dynamic_forms.xml
  - SystemStatus object
  - PagedListInfo object
  - hooks and components
  - HtmlDialog object
  - Lookup Lists, also database table records
  - Action permissions, also database table records
  - Record-level permissions
  - Adding and modifying module specific tables
  - Utils classes
